## Requisitos

1. **PHP** >= 8.1  
2. **Composer** (gestor de dependencias de PHP)  
3. **MySQL** o MariaDB  
4. Terminal o CMD (dependiendo de tu sistema operativo)

> ⚠️ Si no tienes PHP o Composer, los pasos incluyen cómo instalarlos.
